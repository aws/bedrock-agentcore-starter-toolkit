# Amazon Bedrock AgentCore

> Amazon Bedrock AgentCore enables you to deploy and operate highly 
capable AI agents securely, at scale. It offers infrastructure 
purpose-built for dynamic agent workloads, powerful tools to enhance 
agents, and essential controls for real-world deployment. AgentCore 
services can be used together or independently and work with any 
framework including CrewAI, LangGraph, LlamaIndex, and Strands Agents, 
as well as any foundation model in or outside of Amazon Bedrock, 
giving you ultimate flexibility.
